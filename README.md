Bu araç ile Cursor ve Windsurf gibi IDE'leri sınırsız kullanabilirsiniz.

## Derleme

## Paket Kurulumu

Gerekli paketleri yüklemek için aşağıdaki komutu çalıştırın:

```bash
pip install -r requirements.txt
```

### Windows
Projeyi derlemek için:

```bash
.\build.bat
```

### Linux & MacOS
Projeyi derlemek için:

```bash
./build.sh
```
